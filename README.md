# Password-Generator
A Python command-line password generator built to practice randomization, strings, loops, and modular programming.

This project is part of my AI Engineering Learning Roadmap, where each repository focuses on mastering a specific set of Python and software engineering concepts.

---

##  Features

- Generate passwords of any desired length
- Include lowercase letters (a-z)
- Include uppercase letters (A-Z)
- Include numbers (0-9)
- Include special characters
- Shuffle generated passwords for randomness
- Validate password length input
- Generate multiple passwords in a single execution
- Simple command-line interface

##  Technologies Used

- Python 3
- random module
- string module
